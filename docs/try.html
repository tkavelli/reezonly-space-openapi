For "Try it out" to work, add these headers to your API responses:

Access-Control-Allow-Origin: https://tkavelli.github.io Access-Control-Allow-Methods: GET, POST, PUT, DELETE, PATCH, OPTIONS Access-Control-Allow-Headers: Content-Type, Authorization, Accept-Language Access-Control-Allow-Credentials: true

PHP/Yii2 example:

// In your controller or middleware Yii::$app->response->headers->set('Access-Control-Allow-Origin', 'https://tkavelli.github.io'); Yii::$app->response->headers->set('Access-Control-Allow-Methods', 'GET, POST, PUT, DELETE, PATCH, OPTIONS'); Yii::$app->response->headers->set('Access-Control-Allow-Headers', 'Content-Type, Authorization, Accept-Language'); Yii::$app->response->headers->set('Access-Control-Allow-Credentials', 'true'); // Handle OPTIONS preflight requests if (Yii::$app->request->method === 'OPTIONS') { Yii::$app->response->statusCode = 200; return; }
